# Star Citizen ClearShaders

![Logo][FS]

This is a work in progress.

This script is made to clear the shaders cache of Star Citizen at launch.

## Compile

### Windows

```ps1
g++ -o ClearShaders.exe ClearShaders.cpp
```

## Usage

```ps1
ClearShaders.exe
```
